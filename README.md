# Discord Clone

This is a clone of the popular communication platform, Discord, created using HTML and Tailwind CSS. The clone aims to replicate the visual design and layout of Discord's landing page, offering a similar user interface and experience.

## Features

- **Responsive Design**: The project is fully responsive, adapting to different screen sizes.
- **Custom Fonts**: Utilizes custom fonts such as Whitney, Ginto, and ggSans for an authentic look.
- **Tailwind CSS**: Built with Tailwind CSS for a highly customizable and efficient design system.
- **Interactive Elements**: Includes hover effects and animations for buttons and other elements.
- **Gradient Backgrounds**: Implements gradient backgrounds to replicate Discord's dynamic visual style.
- **Embedded Videos**: Uses embedded videos to showcase Discord's features, similar to the original website.

## Tech Stack

- **HTML**: Used for structuring the web page.
- **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.
- **JavaScript**: Minimal usage for Tailwind configuration and responsive behavior.
